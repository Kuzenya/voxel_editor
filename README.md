# SpeechParse

**Voxel Editor** — это кроссплатформенное приложение для создания, редактирования и визуализации трёхмерных сцен, состоящих из вокселей.
Редактор предназначен для обучения основам 3D-моделирования и работы с воксельной графикой.

## Функционал программы

- Интерактивная 3D-визуализация сцены с использованием OpenGL.
- Навигация камеры с помощью клавиатуры (WASD) и мыши.
- Добавление, перемещение, выделение и удаление вокселей.
- Изменение цвета и материалов объектов через редактор материалов.
- Сохранение и загрузка проектов в текстовом формате.
- Кроссплатформенность — работает под Windows, macOS, Linux.
- Минималистичный интерфейс с фокусом на рабочую область.

## Установка и запуск

1. Клонируйте репозиторий:  
```
git clone https://github.com/yourusername/voxel-editor.git
```
2. Установите зависимости:  
```
pip install -r requirements.txt
```
3. Запустите приложение:  
```
python main.py
```

## Использование

### Навигация:
- W, A, S, D — движение камеры.
- Shift + мышь — поворот камеры.
- Escape — выход.

### Работа с объектами:
- N + (X/Y/Z) — добавить воксель и переместить по оси.
- G + (X/Y/Z) — переместить выделенный воксель.
- Delete — удалить выделенный объект.
- O — открыть список объектов.
- C — открыть редактор материалов.

### Файлы:
- M — сохранить сцену.
- Ctrl + M — загрузить сцену.
